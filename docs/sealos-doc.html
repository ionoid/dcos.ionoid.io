# SealOS Secure Linux OS for IoT and Edge devices

SealOS is a secure Linux Operating System tailored for Edge devices. Built by using Yocto Project (opens new window), it is a perfect solution for embedded devices. SealOS includes layers of security protections and ongoing security updates to keep your devices secure and up to date.

SealOS is Open Source and comes with software and security updates.
